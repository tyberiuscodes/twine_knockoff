Welcome

This site is the result of a personal challenge project. I wanted a project to work on in order to learn the ropes with Git and version control, but I also needed something that was sufficiently complex enough to actually warrant using Git. I decided to essentially hand-code a Twine game lookalike using only HTML and CSS (no frameworks), limited to a single page. This isn't meant as a novel idea but was instead inspired by the limitations of petsites such as Subeta, where users have the freedom to customize the individual pages of their "pets" using HTML and CSS. The community has always been extremely creative, but I was especially inspired by the pages of user Tashamon (particularly their pets Mirror and Signal). If you enjoy this project, then I'd encourage you to give those pages a look.
